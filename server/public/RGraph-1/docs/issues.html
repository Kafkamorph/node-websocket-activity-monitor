These are some common issues that you should be aware of. If you're having trouble, you may want to look through these to see if any apply to you or may be the source of your problem.

Opera and context menus
Opera does not support customising the context menu, so the context menu feature attaches the menu to the left mouse button instead of the right. Due to Safari being rather temperamental, this browser also attaches the context menu to the left mouse button.

Annotations aren't saved when running locally in Mozilla Firefox
The solution here is to run the graphs using a web server. At this time (20th November 2009) only Safari 4 and Chrome 4 (Dev) support saving annotations when running locally.
